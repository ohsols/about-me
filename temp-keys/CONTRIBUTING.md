# Contributing

## Ways to Contribute

- **Report Expired Keys** — [Open an issue](../../issues/new?template=report-expired-key.md)
- **Request New Models** — [Open an issue](../../issues/new?template=request-model.md)
- **Add Examples** — Submit a PR to the `examples/` directory
- **Translate** — Help translate README to other languages
- **Improve Docs** — Fix typos, add FAQs, improve guides

## How to Submit

1. Fork → Branch → Change → PR
2. Keep PRs focused, one change per PR
3. Test code examples before submitting

## First Time?

Check issues labeled [`good first issue`](../../labels/good%20first%20issue).
